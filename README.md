# Codex Lens

> 실시간 번역 및 독서 노트 저장을 지원하는 AI 기반 모바일 앱

## 주요 기능
- 카메라 프리뷰 및 실시간 텍스트 인식(OCR)
- 터치 기반 번역 및 오버레이 표시
- 번역/원문 독서 노트 저장 및 목록 조회
- 갤러리 이미지 번역
- 노트별 메모/태그 추가
